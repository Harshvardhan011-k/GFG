### Project Overview
**Financial Saathi** is an AI-powered micro-investment platform designed specifically for first-time users and low-literacy communities. Our goal is to make financial growth accessible to everyone through:
- **Micro-Savings (SIP):** Save as little as â‚¹10/day towards specific life goals (e.g., Education, Festivals).
- **Voice-First AI Chatbot:** Interact with a friendly AI companion in Hindi, Marathi, or English using natural speech.
- **Story-Based Learning:** Earn rewards while learning about finance through relatable local stories.
- **Advisor Portal:** Empower community leaders with AI insights to guide their peers toward financial stability.
- **Premium, Inclusive Design:** A high-end UI designed for simplicity, clarity, and trust.

### Setup and Installation Instructions

#### Backend
1. Navigate to the `backend` folder.
2. Install dependencies: `npm install`
3. Create a `.env` file with `PORT=5000`.
4. Start the server: `npm start`

#### Frontend (Web)
1. Navigate to the `frontend-web` folder.
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000) in your browser.

---

### Usage Instructions
1. **Onboarding:** Choose your preferred language (Hindi, Marathi, or English).
2. **Dashboard:** View your goals and total savings. Click "New Goal" to start a micro-investment.
3. **Chat:** Ask the AI Financial Saathi anything about money via text or voice.
4. **Learn:** Complete short, rewarding stories to improve your financial literacy.
5. **Manage:** Update or delete goals as your financial situation evolves.
